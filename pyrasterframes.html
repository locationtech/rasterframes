RasterFrames in Python

RasterFrames includes Python bindings for using the library from within PySpark. While creating Python documentation on parity with Scala is forthcoming, these resources should help in the meantime:

  • PyRasterFrames README
  • PyRasterFrames Examples
  • RasterFrames Jupyter Notebook

Most features available in the Scala API are exposed in the Python API, and take almost the same form as they do in Scala. Python UDFs on Tile are not yet supported.

If there’s a specific feature that appears to be missing in the Python version please submit an issue so that we might address it for you.
